# EXEMPLO 2: FURADEIRA AUTOMÁTICA
## Introdução
Exemplo simples didático de uma automação implementada na linguagem descritiva

## Descrição do Processo

Desejamos automatizar uma linha de enchimento de garrafas. Devemos ligar a esteira até o sensor
de posicionamento de garrafa ser acionado. Uma vez acionado, devemos desligar a esteira e abrir
a válvula de despejo do líquido por 1 segundos. Em sequência, devemo ligar a esteira novamente
e repetir o processo. 


## Pontos de E/S

 - SD1: Aciona esteira
 - SD2: Abre válvula de enchimento

 - ED1: Sensor de posicionamento da garrafa
